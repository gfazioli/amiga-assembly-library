# CloseInterface

## Synopsis

```asm
CloseInterface(i)
               A0
```

## C Prototype

```c
void CloseInterface(struct Interface *);
```

## Description

Frees all resources allocated by `OpenInterface()`.

## Inputs

- **i** — Pointer to an Interface structure, returned by `OpenInterface()`.

## Result

None.

## See Also

[CloseREI](/docs/rei/close-rei), [OpenInterface](/docs/intui-gadtools/open-interface)

## Implementation

The original 68020 assembly implementation:

```asm
; (20-Jan-1995) --- CloseInterface(i) (a0)
_LVOCloseInterface
        move.l  a6,-(sp)
        move.l  ab_ExecBase(a6),a0
        move.l  ThisTask(a0),a0
        move.l  TC_Userdata(a0),d1
        beq.s   CIExit
        clr.l   TC_Userdata(a0)
        subq.l  #4,d1

        move.l  ab_DosBase(a6),a6
        jsr     _LVOUnLoadSeg(a6)
CIExit  move.l  (sp)+,a6
        rts
```
