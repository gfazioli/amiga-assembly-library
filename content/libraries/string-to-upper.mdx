# StringToUpper

## Synopsis

```asm
StringToUpper(locale, string, len)
              A0      A1      D0
```

## C Prototype

```c
void StringToUpper(struct Locale *locale, STRPTR string, LONG len);
```

## Description

Converts an entire ASCII string to uppercase, in accordance with the localization system.

## Inputs

- **locale** -- Pointer to a Locale structure, or NULL for the default Locale.
- **string** -- Address of the string to convert. It must be null-terminated if you want to pass `len` as NULL.
- **len** -- Number of characters to convert. If NULL, the entire string will be converted.

## Result

None. The string is converted in place.

## See Also

- [StringToLower](/docs/libraries/string-to-lower)
- `locale.library/ConvToUpper()`, `ConvToLower()`

## Implementation

The original 68020 assembly implementation:

```asm
; (26-Nov-1994) --- StringToUpper()
_LVOStringToUpper
        movem.l d2/a4-a6,-(sp)
        move.l  a0,a4
        move.l  ab_Locale(a6),a5
        move.l  ab_LocaleBase(a6),a6
        move.l  d0,d2
        bne.s   SkLen
        moveq   #-1,d2

CalLen  tst.b   (a0)+
        dbeq    d2,CalLen
        neg.l   d2

SkLen   subq.l  #1,d2

TLop    move.b  (a4),d0
        move.l  a5,a0
        jsr     _LVOConvToUpper(a6)
        move.b  d0,(a4)+
        dbeq    d2,TLop

SU_End  movem.l (sp)+,d2/a4-a6
        rts
```
